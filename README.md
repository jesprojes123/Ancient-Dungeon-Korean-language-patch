## 소개

[에이션트 던전 VR](https://erthu.de/ancientdungeon.html) 한글패치입니다.

폰트는 [갈무리9](https://galmuri.quiple.dev/)를 사용합니다.

호환 버전은 ea0.1.6.6(v18083002) 버전입니다.

번역에는 AI+손 번역 복합적으로 사용했습니다.(개인에 따라 부자연스러운 부분 있을 수 있음.)
